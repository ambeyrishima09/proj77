Visual Studio Code



Visual Studio Code is an advanced IDE. It is simple to use and works efficiently. Now, you will firstly think how to download and set it up. So, firstly you need to search "Visual Studio Code download" in your default browser. Then click on the first link you get. Next, hit the download button and choose which type of device you have. As you click on it, it will start downloading. Once it gets downloaded, click on it and select preferable options. You are all done now! You can start uploading folder, creating new files and begin coding!


You can open console screen by right clicking on the output area. Select inspect and click on console. You are there! If you want to print anything on console screen, then you can use console.log() method for it.
